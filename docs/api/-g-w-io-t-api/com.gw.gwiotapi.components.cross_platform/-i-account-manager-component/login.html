login

abstract suspend fun login(hostUserId: String): GWResult<User>

登录认证SDK

不使用技威账号服务时，优先通过这个方法登录认证

App需要实现IHostAccountService，并通过GWIoT.registerHostAccountService(service: IHostAccountService)注册到SDK。

SDK在必要时会调用IHostAccountService中的onRequestUserC2CInfo方法获取账号认证信息。

Return

登录结果

Parameters

hostUserId

App当前登录的用户唯一标识符，仅用于判断SDK缓存的accessToken是否是当前App登录用户，不会上传到技威云。也可以加密后传给SDK。
